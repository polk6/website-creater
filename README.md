# website-creater
基于gulp4的静态网页生成器；适用于简单官网的快速搭建。

# Features
* 支持layout布局
* 支持合并JavaScript、Less
* 支持各页面模块化JS
* 支持LiveReload
* 支持非布局页面(独立页面)，如用户协议

# Usage

## install
```
npm install .
```

## dev
```
npm run dev
```

or 

```
gulp
```

**server**：http://localhost:8083/

## build
```
npm run build
```

or 

```
gulp build
```
